### 问题描述

1. 添加远程仓库地址
   ```
   git remote add [name] [location]
   ```
2. 下载远程仓库信息
   ```
   git fetch [name] [branch]
   ```
3. 查看远程仓库信息
   ```
   git branch -a
   ```
4. (可选)将远程分支检出成本地分支
   ```
   git checkout -b [branch] [name/branch]
   ```
5. 合并分支
   ```
   git merge [branch]
   ```
